psychology 장르 + framework 구조 책 중 신경과학 근거를 활용한 자기계발서의 패턴.

**대표 사례: 뇌는 어떻게 성공하는가 (John B. Arden)**

- hookStrategy: question ("왜 노력해도 안 되는가? → 뇌 배선 때문")
- narrativeArcType: discovery (뇌 작동 원리를 이해하면 삶이 바뀐다는 발견의 여정)
- emotionalTone: disciplined + hopeful (과학 권위 기반 규율 + 신경가소성이 주는 희망)
- urgencyLevel: medium (지금 시작하면 뇌가 바뀐다, 린치핀처럼 극단적 긴박감은 없음)
- contentMode: mixed (과학적 개념 설명 + 실행 가능한 조언 혼합)
- uniqueElements: 신경망 연결 다이어그램, before/after 뇌 배선 시각화, 방사형 레이더 차트

**selfHelp framework 유형 B(시스템/원리 중심)와의 차이:**

- selfHelp 유형 B (Atomic Habits): 수학적 논거, 순환 다이어그램, 정체성 레이어
- psychology neuroscience: 뇌과학 연구 인용, 신경망 시각화, before/after 뇌 변화

**keyConceptCount 결정 방법:**

- FEED처럼 명시적 약어 프레임워크: 약어 요소 수 (FEED = 4)
- 약어 없이 챕터 기반이면: 핵심 영역 수 (보통 4~6)

**Why:** 뇌과학 책은 과학적 권위와 희망(신경가소성) 두 축이 공존하므로 disciplined + hopeful 조합이 적합. question hook은 독자가 이미 품고 있는 의문("왜 안 될까")을 직접 건드려 즉시 공감을 얻는다.

**How to apply:** 저자가 신경과학/뇌과학 배경을 가진 경우 genre를 psychology로 분류하고, FEED처럼 영문 약어 프레임워크가 있으면 coreFramework에 풀네임 병기. visualMotifs에 neural-network 반드시 포함.
