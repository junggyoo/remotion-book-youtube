framework 구조의 selfHelp 책이라도 핵심 주장이 "통념 뒤집기"일 때 hookStrategy는 transformation이 아닌 contrarian을 선택한다.

**Why:** Atomic Habits의 핵심은 "목표를 버려야 목표에 도달한다 (시스템 > 목표)"는 역설이다. 단순한 변화 사례 제시(transformation)보다 통념 반박(contrarian)이 시청자의 주의를 더 강하게 잡는다.

**How to apply:** 책의 메인 주장이 다음 패턴을 따를 때 contrarian을 우선 고려한다:

- "X가 아니라 Y가 답이다" (X = 일반적 통념)
- 수학적/데이터 기반 반박이 있을 때
- 독자가 이미 X를 시도하고 실패했을 가능성이 높을 때

반면 저자의 극적 개인 경험이 책의 핵심 서사일 때는 transformation을 유지한다 (예: 미라클 모닝).
